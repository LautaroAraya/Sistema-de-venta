# ====================================================================
#                    CÓMO USAR EL SISTEMA
# ====================================================================

DESARROLLO:
1. Instalar Python 3.8+
2. pip install -r requirements.txt
3. python inicializar_datos.py (primera vez)
4. python main.py

O simplemente ejecutar: run.bat

GENERAR EJECUTABLE:
1. Ejecutar: build_exe.bat
2. El .exe estará en dist/SistemaVentas.exe

CREAR INSTALADOR:
1. Instalar Inno Setup
2. Compilar installer_script.iss
3. El instalador estará en installer/SistemaVentas_Setup.exe

USUARIO FINAL:
1. Ejecutar SistemaVentas_Setup.exe
2. Seguir el asistente de instalación
3. Ejecutar desde el acceso directo

# ====================================================================
#                    FLUJO DE TRABAJO DE UNA VENTA
# ====================================================================

1. Login → Validar credenciales
2. Menú Principal → Seleccionar "Nueva Venta"
3. Ingresar datos del cliente (opcional)
4. Buscar producto → Aparece en tiempo real
5. Seleccionar producto → Muestra precio y stock
6. Ingresar cantidad y descuento
7. Agregar a la venta → Se muestra en tabla
8. Repetir pasos 4-7 para más productos
9. Verificar totales
10. Procesar venta → Genera factura
11. Opción de generar PDF
12. Stock se actualiza automáticamente
13. Venta registrada en reportes

# ====================================================================
#                    LIMITACIONES Y CONSIDERACIONES
# ====================================================================

- SQLite tiene un límite teórico pero soporta millones de registros
- Para uso empresarial grande, considerar migrar a MySQL/PostgreSQL
- Las facturas PDF no son editables (solo lectura)
- Un usuario puede estar logueado desde múltiples PCs
- El sistema es local (no tiene servidor centralizado)
- Recomendado para pequeñas y medianas empresas
